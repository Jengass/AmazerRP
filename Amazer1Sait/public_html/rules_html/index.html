Правила Amazer RP

// 1. Основные RP-Правила
  • 1.1. Non-RP (Нон-РП): Действия, невозможные в реальной жизни (бег по крышам машин, игнорирование аварий, езда на сломанном авто).Kick (первый раз) / Ban 1 день (рецидив)
  • 1.2. FearRP (Фар-РП): Отсутствие страха за жизнь персонажа. Например, если на тебя наставили 3 пушки, ты обязан подчиняться, а не убегать.Kick / Ban 6-12 часов
  • 1.3. Leave RP (LRP): Выход из игры во время РП-ситуации (ареста, ограбления, допроса).Ban 1-3 дня
  • 1.4. MetaGaming (MG): Использование чата для передачи инфы из реала (типа "привет, я из дискорда").Mute 15-30 мин
// 2. Правила убийств и боя
  • 2.1. RDM (Random DeathMatch): Убийство игрока без какой-либо РП причины. Просто подошел и застрелил/зарезал.Ban 1 день
  • 2.2. VDM (Vehicle DeathMatch): Наезд на игрока машиной с целью убить или помешать РП.Kick / Ban 12 часов
  • 2.3. Spawn Kill (SK): Убийство игроков на месте их появления (спавне).Ban 3 дня
// 3. Общение и поведение
  • 3.1. Оскорбление родных: Строжайше запрещено в любой форме.Ban 15-30 дней
  • 3.2. Обман администрации: Вранье в репорт о нарушениях других игроков.Ban 1-7 дней
  • 3.3. Багоюз: Использование багов Роблокса (проход сквозь стены, полет и т.д.).P-Ban (Перманентный бан)
  • 3.4. Non-Rp розыск: Розыск, выданный не по правильной причине/по приколу/странная причина.Ban 1 день
  • 3.5. Розыск неразрешенной командой: Розыск выдал гражданин (Исключение: команды Полиция, ФСБ, ГУМВД, УФСБ).Ban 2-3 дня
// 4. Правила государственных организаций (ПГО)
  • 1.1. ПГО: Затягивание RP процесса: Сотрудник, задержавший вас, должен отвезти в ближайший участок. Если он везет в дальний — это затягивание.Ban 5-12 часов
📌 Правила могут обновляться без предупреждения игроков.
